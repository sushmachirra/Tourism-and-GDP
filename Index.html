GDP and Tourism

In this website you are going to visualize the project "GDP and Tourism". This is a data visualization project, in this we are going to visualize the tourism data and will be finding the relation between attributes. We will be visualizing the data in different idioms like bar charts, line chart, scatter plots and map. We will be using different visualization platforms to visualize and analyze the data.

We have visualized data in 3 different tools namely: Tableau, Vizhub and Python

We mainly visualized data in these types of idioms namely: Map, Scatterplot, Bar chart, Stacked Bar chart, Coxcomb,Bubble chart, line chart and Pie chart
